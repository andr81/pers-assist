# Недельное планирование

Проведи сессию недельного планирования:

## Шаги

1. **Ревью прошлой недели**
   - Спроси пользователя о главных достижениях
   - Спроси что не удалось сделать и почему
   - Узнай какие уроки извлечены

2. **Цели на неделю**
   - Помоги определить 3-5 главных целей на неделю
   - Для каждой цели определи конкретные задачи
   - Создай задачи в SingularityApp (create_task)

3. **Распределение по дням**
   - Помоги распределить задачи по дням недели
   - Учитывай загруженность и энергию

4. **Области жизни**
   - Проверь, что задачи покрывают важные области:
     - Работа/карьера
     - Здоровье/фитнес
     - Обучение
     - Социальное
     - Личное

## Формат ответа

- Используй таблицы для наглядности
- Группируй задачи по дням
- Отмечай приоритеты
