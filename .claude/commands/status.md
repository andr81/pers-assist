# Статус по всем областям

Покажи текущий статус по всем областям жизни пользователя:

## Области для проверки

### 1. Личная эффективность
- Текущие задачи в работе
- Прогресс по привычкам
- Уровень продуктивности

### 2. Планирование
- Задачи на сегодня/неделю
- Предстоящие дедлайны
- События в календаре

### 3. Финансы
- Статус бюджета (когда будет интеграция)
- Запланированные расходы

### 4. Здоровье
- Привычки связанные со здоровьем
- Качество сна, питание

### 5. Фитнес/спорт
- Тренировки за неделю
- Прогресс по целям

### 6. Обучение
- Текущие курсы/книги
- Прогресс обучения

### 7. Социальное
- Запланированные встречи
- Контакты для связи

## Инструкции

1. Используй доступные MCP инструменты для получения данных
2. Для недоступных областей спроси пользователя
3. Выдели области требующие внимания
4. Предложи конкретные действия

## Формат

- Используй индикаторы статуса (хорошо/нормально/требует внимания)
- Будь лаконичен
- Выдели главное
