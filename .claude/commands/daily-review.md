# Ежедневный обзор

Проведи ежедневный обзор для пользователя:

## Шаги

0. **Текущая дата и время**
   - Получи текущее время через get-current-time
   - Это нужно для корректной фильтрации задач и событий

1. **Задачи на сегодня**
   - Получи просроченные задачи через list_tasks (startDateTo вчера, includeCompleted=false)
   - Получи задачи на сегодня через list_tasks (startDateFrom/To на сегодня)
   - Покажи задачи, сгруппированные по статусу: просроченные, на сегодня

2. **Привычки**
   - Получи список привычек (list_habits)
   - Покажи какие привычки нужно выполнить сегодня

3. **Обзор календаря**
   - Получи события на сегодня из календарей: primary, family
   - Используй list-events с timeMin/timeMax на сегодня
   - Покажи время и название событий

4. **Фокус дня**
   - Спроси пользователя о главном фокусе на день
   - Помоги определить 3 главные задачи

## Формат ответа

Структурируй ответ с эмодзи для наглядности:
- Используй чекбоксы для задач
- Выделяй важное жирным
- Будь лаконичен
