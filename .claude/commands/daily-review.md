# Ежедневный обзор

Проведи ежедневный обзор для пользователя:

## Шаги

1. **Задачи на сегодня**
   - Получи список задач на сегодня через SingularityApp (get_today_tasks)
   - Покажи задачи, сгруппированные по приоритету

2. **Привычки**
   - Получи список привычек (list_habits)
   - Покажи какие привычки нужно выполнить сегодня

3. **Обзор календаря** (когда будет интеграция)
   - Покажи события на сегодня

4. **Фокус дня**
   - Спроси пользователя о главном фокусе на день
   - Помоги определить 3 главные задачи

## Формат ответа

Структурируй ответ с эмодзи для наглядности:
- Используй чекбоксы для задач
- Выделяй важное жирным
- Будь лаконичен
