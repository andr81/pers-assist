# Ежедневный вечерний обзор

Проведи ежедневный вечерний обзор для пользователя:

## Шаги
- Получи список выполненных сегодня задач через SingularityApp
- Получи события и встречи из всех google-календарей
- По сегодняшним выполненном задачам и встречам сделай список возможных туду, по которым нужно создать задачи на будущее.
- Составь список задач которые остались не выполнены.
- Получи список задач singularity которые нужно выполнить завтра.
- Получи встречи и события из всех google-календарей на завтра
- Составь план с советами на завтрашний день
- Получи прогноз погоды на завтра в Минске и кратко его опиши
- Получи список привычек singularity и предложи какие нужно заполнить

## Формат ответа

Структурируй ответ с эмодзи для наглядности:
- Используй чекбоксы для задач
- Выделяй важное жирным
- Будь лаконичен
