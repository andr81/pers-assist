# Субагенты персонального ассистента

Планируемые специализированные агенты для каждой области жизни.

## Концепция

Каждый агент специализируется на своей области и имеет доступ к соответствующим MCP инструментам и знаниям.

## Планируемые агенты

### 1. Productivity Agent
**Область:** Личная эффективность

**Задачи:**
- Анализ продуктивности
- Рекомендации по улучшению фокуса
- Работа с привычками
- Оптимизация рабочих процессов

**MCP интеграции:** SingularityApp

---

### 2. Planning Agent
**Область:** Планирование

**Задачи:**
- Помощь в постановке целей
- Декомпозиция больших задач
- Планирование дня/недели/месяца
- Управление дедлайнами

**MCP интеграции:** SingularityApp, Google Calendar

---

### 3. Finance Agent
**Область:** Финансы

**Задачи:**
- Анализ расходов
- Планирование бюджета
- Отслеживание финансовых целей
- Напоминания о платежах

**MCP интеграции:** (в планах)

---

### 4. Health Agent
**Область:** Здоровье

**Задачи:**
- Отслеживание сна
- Мониторинг питания
- Напоминания о медицинских визитах
- Рекомендации по здоровью

**MCP интеграции:** (в планах)

---

### 5. Fitness Agent
**Область:** Фитнес/спорт

**Задачи:**
- Планирование тренировок
- Отслеживание прогресса
- Рекомендации по восстановлению
- Мотивация

**MCP интеграции:** (в планах)

---

### 6. Learning Agent
**Область:** Обучение

**Задачи:**
- Отслеживание прогресса по курсам
- Управление списком книг
- Создание конспектов
- Планирование обучения

**MCP интеграции:** Notion/Obsidian

---

### 7. Social Agent
**Область:** Социальное

**Задачи:**
- Управление контактами
- Напоминания о встречах
- Отслеживание важных дат
- Нетворкинг

**MCP интеграции:** Google Calendar, (CRM в планах)

## Реализация

Агенты будут реализованы как:
1. Slash-команды с детальными промптами
2. Или отдельные конфигурации субагентов Claude Code

Приоритет реализации определяется по мере развития проекта.
